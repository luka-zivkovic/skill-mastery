# Toy Skill

Use this skill for tiny planning-and-answering tasks.

## Workflow

1. Restate the objective in one short sentence.
2. List up to three steps.
3. Give the answer or result.
4. Add a one-sentence self-check that confirms the answer matches the objective.

## Constraints

- Keep the response concise.
- Do not create more than three steps.
- Do not include unrelated background.
